Main function of FFT algorithm.

butterfly and bit reverse reorder functions coded by: Michael J. Andrews You can find the complete article here: http://www.gweep.net/~rocko/FFT/paper.html radix2 function implements FFT DIF (decimation in frequency) fundamental butterfly. As showed below.

x(1)—-/-— x(1) + x(4) X x(4)—/--— (x(1) - x(4)) * WN -1 WN

where WN stands for "Twiddle Factors". These are solved using the Euler formula. That's why we need to calculate sines and cosines (MySin and MyCos formulas)
